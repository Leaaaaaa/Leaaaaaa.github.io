appium之微信点赞

发表于 2019-11-20
  • 这是一个自动化测试微信朋友圈点赞的简单代码。本来想要给两个动态点赞，无奈太菜，无法定位所有属性都相同的评论元素，只能点出现的第一个评论框。尝试过用绝对路径，但是上下两个评论框的很多父节点都相同，多次尝试无果。查了一些资料也没找到想要的解决方案，只有先留着这个问题以后去解决啦。
  • 下面是实现代码

`from appium import webdriver
import time
from selenium.webdriver.support.wait import WebDriverWait

desired_caps = dict()
desired_caps[‘platformName’] = ‘Android’
desired_caps[‘platformVersion’] = ‘6.0.1’
desired_caps[‘deviceName’] = ‘test’
desired_caps[‘appPackage’] = ‘com.tencent.mm’
desired_caps[‘appActivity’] = ‘.ui.LauncherUI’

driver = webdriver.Remote(‘http://localhost:4723/wd/hub', desired_caps)

打开微信界面

点击发现//显示等待

wait = WebDriverWait(driver, 15).until(lambda x: x.find_element_by_xpath(“//*[@text=’发现’]”))
wait.click()

time.sleep(2)

点击朋友圈

driver.find_element_by_xpath(“//*[@text=’朋友圈’]”).click()
time.sleep(2)

滑动屏幕

driver.swipe(866, 1364, 866, 684, 3000)
time.sleep(2)

点击评论按钮

driver.find_element_by_xpath(“//*[@content-desc=’评论’]”).click()
time.sleep(2)

点赞

driver.find_element_by_xpath(“//*[@text=’赞’]”).click()

time.sleep(5)

driver.quit()`
