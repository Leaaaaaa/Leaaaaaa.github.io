微信朋友圈测试用例

发表于 2019-11-21

朋友圈测试方案

动态

界面

  • 朋友圈界面是否符合设计
  • 朋友圈背景图片是否正确显示

功能

  • 自己的动态

    • 查看权限

      • 发表动态所有人可见，好友是否可见
      • 发表仅自己可见动态，朋友圈是否显示
      • 发表分组可见动态，不在可见组的人是否能看到

    • 查看方式

      • 从朋友圈动态看

        • 显示照片视频文字

      • 从我的相册看

        • 显示照片，视频

    • 动态发布方式

      • 从相册选择

        • 是否能完整访问相册内容

      • 视频类型

        • 拍摄

          • 是否能正常访问相机功能
          • 视频是否完整显示

      • 长按

        • 是否能跳到发表纯文案页面

    • 无网络下发表动态

    • 提醒谁看是否成功

    • 所在位置是否正确显示

  • 动态内容

    • 图片或文案类型

      • 图片显示是否正确以及图片的最大数量限制
      • 不同数量的图片排版是否符合预期
      • 文案字数的最大限制
      • 文案的特殊字符，表情，链接等是否正确显示

  • 别人的动态

    • 消息提醒

      • 发现是否有红点
      • 朋友圈是否有头像

    • 用户屏蔽了朋友圈所有提醒，是否屏蔽成功

    • 浏览朋友圈过程中有新动态是否有提醒

    • 点进动态发布人的头像能否跳转资料页

    • 从资料页进入该用户的朋友圈能否看到动态记忆历史动态

    • 好友设置动态显示期限是否显示正确

    • 屏蔽好友动态后是否显示

    • 屏蔽好友动态后是否提醒

    • 好友动态删除，是否有动态提醒

    • 好友动态已查看，提醒是否消失

    • 动态人昵称

      • 未备注，显示昵称
      • 备注，显示备注
      • 是否显示特殊字符

并发

  • 短时间内多个好友发表动态

性能

  • 网络延迟，断网是否发表
  • 一定时间内多次发表动态
  • 刚发表动态时就删除动态

兼容

  • 不同手机操作系统
  • 不同微信版本
  • 不同语言
  • 不同分辨率设备
  • 不同型号的一定设备

点赞

界面

  • 界面是否简介美观
  • 是否符合设计

功能

  • 点赞人

    • 点赞是否实时显示结果

    • 点赞后展示的是不是自己的昵称

    • 点赞后是否能取消

    • 取消后，重新点赞

    • 点赞后，其他共同好友点赞，是否有消息提醒

    • 点赞后，非共同好友点赞，是否有消息提醒

    • 点击点赞人昵称，是否能跳转到其资料页

    • 能否看到其他人的点赞

      • 共同好友
      • 非共同好友
      • 发动态的用户本人点赞

    • 从朋友圈动态点赞

    • 访问好友相册，对历史动态点赞

    • 登录已失效点赞

  • 被点赞人

    • 自己给自己点赞

    • 点赞人昵称

      • 设置备注的好友，展示备注名
      • 未设置备注好友，展示用户昵称
      • 用户昵称含有特殊符号，能够显示
      • 点赞用户中有相同昵称
      • 多个好友点赞，换行显示

    • 从朋友圈动态页查看

    • 从我的相册查看

    • 解除好友关系后是否还显示其点赞

    • 拉黑后是否显示其点赞

  • 点赞内容

    • 图片
    • 文案
    • 视频
    • 链接

  • 消息提醒

    • 朋友圈右上角

      • 多条
      • 一条

    • 点击朋友圈，页面

    • 进入朋友圈，顶端显示

    • 浏览朋友圈过程中，有人点赞，刷新是否显示消息提醒

    • 用户屏蔽了朋友圈消息提醒是否屏蔽成功

    • 消息详情列表显示

    • 取消点赞后是否提醒

    • 先点赞后，查看后又点赞，是否提醒

    • 消息未查看，换设备登录

    • 消息已查看，换设备登录

  • 是都有最多点赞人数限制

  • 点赞人是否按时间先后排序

并发

  • 多个用户同时点赞
  • 连续多次点赞
  • 用户点赞未提交成功时，删除动态

兼容

  • 不同手机操作系统
  • 不同微信版本
  • 不同语言
  • 不同分辨率设备
  • 不同型号的一定设备

性能

  • 网络延迟，断网

    • 响应延迟
    • 丢包

  • 压力测试

    • 一定时间内多个用户点赞
